## Design Context

Outlay is a calm, exact **subscription tracker** (Next.js + Convex, bilingual
cs/en). Design register is **product** (the dashboard), with the marketing
landing as an equally-valued **brand** surface.

- **`PRODUCT.md`** — strategy: users, purpose, brand personality (calm, exact,
  dependable), anti-references, design principles, accessibility (WCAG 2.1 AA+).
- **`DESIGN.md`** + **`.impeccable/design.json`** — the visual system ("The Quiet
  Ledger"): Ink-Navy-on-Paper with one Ledger-Green accent, amber for attention
  only, all money in Geist Mono tabular numerals, flat-by-default surfaces.

Read both before designing or building UI so new screens stay on-brand.
