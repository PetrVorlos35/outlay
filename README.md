# Outlay

A full-stack app built with:

- [Convex](https://convex.dev/) for the backend (database, server logic)
- [Next.js](https://nextjs.org/) for the frontend and hosting
- [Tailwind](https://tailwindcss.com/) for UI
- [Convex Auth](https://labs.convex.dev/auth) for authentication

## Get started

```
npm install
npm run dev
```

## Project layout

- `app/` — Next.js routes and components
- `convex/` — Convex backend functions and schema
- `components/` — shared React components
